# Superstore Sales Dashboard

## Project Overview

This Power BI dashboard provides an interactive analysis of Superstore sales performance, profitability, orders, returns, and customer purchasing trends. The dashboard helps identify high-performing categories, profitable regions, and business areas requiring improvement.

## Key Metrics

## Business Insights

* Technology category generated the highest sales.
* West region contributed the highest profit.
* Copiers emerged as the most profitable sub-category.
* Tables showed negative profitability and require attention.
* Profit increased consistently between 2014 and 2017.

## Features

* Interactive filters for Category, Region, and Sub-Category.
* KPI Cards for Sales, Profit, Orders, Returns, and Profit Margin.
* Sales and Profit Trend Analysis.
* Regional Profit Distribution.
* Category-wise Sales Performance.
* Sub-Category Profit Analysis.
