**Super Store Sales Dashboard**

Hey everyone, this is a Power BI project I've been working on to analyze the sales performance of a super store.

**About the Data**

This dataset contains information about orders, customers, product shipping details, and key metrices like sales, profit, and discount amounts

**Key Insights**

Here's a qucik look at what the dashboard shows:

1. Total Sales:22.64K
   
2. Total Profit:8.40k
   
3. Total Orders:3312
   
4. profit Margin:13%

You can also break down profits by region, see which categories perform best, and check out the profit trends over time. 

**"To view the dashboard"
**
1. Download the .pbix file from this repository.
2. Open the downloaded file using Power BI Desktop.
